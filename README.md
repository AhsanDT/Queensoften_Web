

## About Installation

### Server Requirement
- PostegreSQL 15
- PHP 8.0

### About Installation
Use the .env.example file for env variables.
Create a database into postgresql with the name of queens-of-ten

After this Run the following commands on the project terminal
- php artisan migrate.
- php artisan db:seed.
